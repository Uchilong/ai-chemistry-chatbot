# chatbot-ai-chemistry
wil be used to compete
in the AI Chemistry Hackathon.

## Features
- **Chemical Formula Parsing**: Understands complex molecular structures.
- **Reaction Prediction**: Predicts products for organic and inorganic reactions.
- **Safety Guidelines**: Provides safety data and handling instructions for various compounds.

## Tech Stack
- **Language**: Python
- **Framework**: FastAPI / Flask
- **AI Model**: OpenAI GPT-4 / Custom LLM
- **Database**: Vector database for chemical documentation retrieval

## Getting Started
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python main.py`
